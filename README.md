Бот для записи в барбершоп через Телеграм

Стек (Python 3.14 + aiogram)

В боте есть возможность посмотреть адрес, услуги, часы работы и оформить запись

Как запустить - создать файл .env, вписать BOT_TOKEN, pip install -r requirements.txt, написать в терминал python3 main.py
